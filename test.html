Teqplay
Ik heb stage gelopen bij het bedrijf Teqplay. Teqplay is een IT bedrijf dat opgericht is door Leon Gommans en Richard van Klaveren en het bevindt zich in de Van Nellefabriek in Rotterdam. Het bedrijf houdt zich bezig met havenlogistiek. Teqplay levert oplossingen voor de logistieke markt, die verwerkt worden in applicaties en websites.

Stageopdracht
Mijn stage opdracht luidt als volgt:
Teqplay krijgt data binnen via verschillende data sources, zoals AIS of BAS. Deze data sources bevatten allemaal data over schepen. Data over eenzelfde schip kan verschillen over verschillende data sources, wat kan zorgen voor conflicterende data. Dit zorgt voor ‘spook schepen’ die verwerkt worden in de applicaties en websites van Teqplay. Spook schepen zijn schepen die niet echt bestaan, maar doordat er incorrecte data is opgeslagen in de database verschijnen deze toch in het platform van Teqplay. Teqplay heeft nog geen database waar alle gegevens van de schepen correct zijn. Mijn opdracht is om deze database te creëren, door middel van verschillende data sources te combineren tot één data source. Voor dat deze data gecombineerd gaat worden, zal de data gevalideerd worden op correctheid door middel van een ‘data checker’. Deze ‘data checker’ zal eerst door mij gemaakt moeten worden. Vervolgens zal de correcte data opgeslagen worden in een database die uiteindelijk gebruikt zal worden als uitgangspunt voor Teqplay. Het project zal in de programmeertaal Kotlin geschreven worden met als framework Spring Boot en de data zal opgeslagen worden met behulp van API calls in een database genaamd MongoDB. Er wordt gebruik gemaakt van sprints die 2 weken duren. Na elke sprint zal er een feedback moment zijn waarbij de volgende stap van de opdracht besproken wordt.

Leerdoelen
Op mijn stageopdracht heb ik twee leerdoelen gebaseerd, namelijk:
  • Tijdens mijn stage bij Teqplay wil ik meer kennis opdoen over API calls en databases. Dit wil ik bereiken door een database te gebruiken waar ik nog nooit mee gewerkt heb, zodat ik kennis heb over verschillende databases. Om mijn leerdoel te kunnen bereiken, moet ik mijzelf informeren over MongoDB en hoe de API calls werken in samenwerking met MongoDB. De opdracht is vooral gericht op data, waardoor ik veel gebruik zal maken van API calls en zo hier meer kennis over zal opdoen. Ik heb mijn leerdoel behaald als ik een volledige Mongo database heb kunnen opstellen en heb kunnen gebruiken voor Mijn stageopdracht. Dit ga ik aantonen door middel van screenshots van de code.
  • Tijdens mijn stage bij Teqplay wil ik een nieuwe programmeertaal leren. Dit wil ik behalen doordat de programmeertaal Kotlin te gebruiken bij mijn stageopdracht. Om mijn leerdoel te kunnen behalen, moet ik mijzelf informeren over Kotlin om uiteindelijk de informatie toe te passen in mijn stageopdracht. Ik heb mijn leerdoel behaald als ik mijn stageopdracht heb gemaakt in de programmeertaal Kotlin. Dit ga ik aantonen door middel van screenshots van de code.
